Project Overview

WARTE2030 develops novel Multi-Agent Reinforcement Learning (MARL) systems for autonomous decision-making in safety-critical power grid infrastructure. the power grid application is the beachhead â€“ our mission is to build trustworthy autonomous systems for European eletricity infrastructure. Within the WARTE2030 project we set the goal of having one autonomous agent deployed in a control room by 2030.

Challenge: Electric power grids face increasing complexity from decentralized renewable sources and infrastructure bottlenecks. Legacy control systems rely on offline, time-intensive optimization methods that cannot meet real-time operational demands.

We enable real-time congestion management through hierarchical MARL coordination embedded in cloud-native control center software, combining sub-second response times with regulatory auditability and human-supervised autonomy.

System Demonstration

Watch our MARL orchestrator coordinate three agents to resolve a simulated transmission grid congestion on a realistic network topology. The system demonstrates end-to-end operation from grid telemetry ingestion through Kafka event streams to explainable operator recommendations.

System Architecture

Our MARL orchestration implements Centralized Training with Decentralized Execution (CTDE) in a zero-trust microservice architecture:

Strategic Layer

Day-ahead congestion planning with 24-hour horizon optimization

Tactical Layer

Regional coordination and contingency response (minutes to hours)

Operational Layer

Real-time control execution for topology reconfiguration and asset control

Kafka Event Stream

Stateless, air-gapped communication backbone

Anti-Corruption Layer

Zero-trust security boundary and legacy system integration

HITL Gateway

Human-in-the-loop oversight with confidence-based approval thresholds
